1. Кто собирает данные

Приложение разработано компанией [ИП/ООО "Твоя компания"] (далее — Разработчик).

2. Какие данные мы собираем

Приложение может собирать следующую информацию:

  • IMEI, Android ID, рекламный идентификатор (AAID) — для аналитики и предотвращения мошенничества
  • Модель устройства и версия ОС — для оптимизации работы
  • IP-адрес — для геолокации (с вашего разрешения)
  • Данные о действиях в приложении — для улучшения функционала

3. Зачем мы собираем данные

  • Для работы приложения
  • Для аналитики (Huawei Analytics, AppMetrica)
  • Для показа рекламы

4. Передача третьим лицам

Мы передаём данные следующим сервисам:

  • Huawei — Push-уведомления, аналитика
  • Яндекс.Метрика / AppMetrica — аналитика
  • AdMob / Huawei Ads — показ рекламы

5. Ваши права

Вы можете в любой момент:

  • Удалить аккаунт — через настройки приложения
  • Запросить полную выгрузку ваших данных — напишите на email
  • Отозвать разрешения — в настройках телефона
